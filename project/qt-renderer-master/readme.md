# QtRenderer

#### 介绍
C++编写的软光栅，实现blinn-phong光照模型，法线和纹理贴图，齐次裁剪，背面剔除，纹理双线性插值，场景管理，相机控制。

上述代码定义了顶点（v）、纹理坐标（vt）、法线（vn）和面（f）。面定义了哪些顶点、纹理坐标和法线组合在一起形成物体的表面。
